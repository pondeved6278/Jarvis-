# Jarvis
Jarvis - Voice Assistant Using Python



Jarvis is an AI-based desktop voice assistant for Windows, which can perform day-to-day tasks through voice commands. It combines various features like voice recognition, text-to-speech, system functionality, and the use of various web interfaces to act as an ideal voice assistant.



**Features**

Input and Output via Voice Interface

Searching Wikipedia using speech output

Open system applications (Notepad, Calculator, Camera, CMD, Discord)

Weather Info Using OpenWeather API

Current news headlines

Jokes and random advice

IP address lookup

WhatsApp Auto Messaging

Sending Emails Using the Gmail App Passwords



**Technologies Used**

Python 3.11+

SpeechRecognition

pyttsx3

requests

Wikipedia

pywhatkit

**Project Purpose**

This project is developed for the purpose of learning automation programming using Python with APIs and voice interaction with the desktop environment.
